Lazymux

📘 Apa itu Lazymux?

Lazymux adalah tool installer untuk Termux yang dirancang untuk memudahkan pengguna dalam menginstal berbagai macam tool hacking dan penetration testing. Lazymux menyediakan banyak kategori tool, seperti information gathering, vulnerability analysis, exploitation tools, dan lain-lain. Dengan Lazymux, kamu bisa dengan cepat membangun lingkungan penetration testing di Termux tanpa harus menginstal setiap tool secara manual.

🛠️ Instalasi

Termux

pkg update && pkg upgrade -y
pkg install git python -y
git clone https://github.com/Gameye98/Lazymux
cd Lazymux
python lazymux.py

Linux (Ubuntu/Debian)

Informasi: Lazymux dirancang khusus untuk Termux dan tidak tersedia untuk Linux. Kamu bisa menginstal tool-tool hacking secara manual di Linux menggunakan `apt`.

🔍 Penggunaan Dasar (Termux)

Setelah menjalankan `python lazymux.py`, kamu akan melihat menu Lazymux. Pilih kategori tool yang ingin kamu install, lalu pilih tool yang diinginkan. Lazymux akan secara otomatis mengunduh dan menginstal tool tersebut.

Beberapa tool yang tersedia di Lazymux:

  • Nmap
  • Sqlmap
  • Metasploit
  • Hydra
  • John the Ripper
  • dan banyak lagi!
