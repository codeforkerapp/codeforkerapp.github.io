Zphisher

📘 Apa itu Zphisher?

Zphisher adalah tool phishing otomatis yang memiliki lebih dari 30 template untuk berbagai platform. Tool ini mudah digunakan dan dapat dijalankan di Termux maupun Linux.

🛠️ Instalasi

Termux

apt update && apt upgrade -y
apt install git php openssh curl -y
git clone https://github.com/htr-tech/zphisher
cd zphisher
chmod +x zphisher.sh
bash zphisher.sh

Linux (Ubuntu/Debian)

sudo apt update
sudo apt install git php openssh curl -y
git clone https://github.com/htr-tech/zphisher
cd zphisher
chmod +x zphisher.sh
bash zphisher.sh

🔍 Penggunaan Dasar

Menjalankan Zphisher

bash zphisher.sh

Ikuti petunjuk di layar untuk memilih template phishing, mengkonfigurasi opsi, dan memulai server phishing.

⚠️ Peringatan Penting

Perhatian! Zphisher adalah alat yang powerful. Gunakan dengan bijak dan bertanggung jawab.

  • Pastikan Anda memiliki izin sebelum membuat atau menyebarkan halaman phishing.
  • Jangan gunakan Zphisher untuk aktivitas ilegal.
  • Gunakan Zphisher hanya untuk tujuan pengujian keamanan dan pendidikan.

Penyalahgunaan Zphisher dapat melanggar hukum dan menyebabkan kerugian bagi orang lain!
