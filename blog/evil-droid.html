Evil-Droid

üìò Apa itu Evil-Droid?

Evil-Droid adalah tool untuk membuat payload Android dan menyisipkannya ke dalam aplikasi Android yang sah. Tool ini memungkinkan kamu untuk membuat aplikasi Android berbahaya yang dapat digunakan untuk mengontrol perangkat korban dari jarak jauh. Evil-Droid dapat digunakan untuk berbagai tujuan, seperti mencuri data, memata-matai aktivitas pengguna, dan mengambil alih perangkat.

Termux

Informasi: Evil-Droid tidak tersedia di repositori paket Termux. Kamu perlu menginstalnya secara manual dengan mengunduh source code dari GitHub.

Instalasi Manual di Termux

pkg update && pkg upgrade -y
pkg install git python -y
git clone https://github.com/M4sc3r4n0/Evil-Droid.git
cd Evil-Droid
pip install -r requirements.txt

Linux (Ubuntu/Debian)

Informasi: Evil-Droid tidak tersedia di repositori paket Ubuntu/Debian. Kamu perlu menginstalnya secara manual dengan mengunduh source code dari GitHub.

Instalasi Manual di Linux

sudo apt update
sudo apt install git python3 python3-pip -y
git clone https://github.com/M4sc3r4n0/Evil-Droid.git
cd Evil-Droid
pip3 install -r requirements.txt
