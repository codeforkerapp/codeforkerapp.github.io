XSStrike

📘 Apa itu XSStrike?

XSStrike adalah advanced Cross Site Scripting (XSS) detection suite yang dilengkapi dengan berbagai fitur canggih, seperti intelligent payload generator, powerful fuzzing engine, dan fast crawler. XSStrike dapat mendeteksi berbagai jenis XSS vulnerability, termasuk reflected dan DOM-based XSS.

🛠️ Instalasi

Termux

pkg update && pkg upgrade -y
pkg install python3 git -y
git clone https://github.com/s0md3v/XSStrike.git
cd XSStrike
pip3 install -r requirements.txt

Linux (Ubuntu/Debian)

sudo apt update
sudo apt install python3 python3-pip git -y
git clone https://github.com/s0md3v/XSStrike.git
cd XSStrike
pip3 install -r requirements.txt

🔍 Penggunaan Dasar

Menjalankan XSStrike

python3 xsstrike.py -u url

Keterangan:

  • -u: URL target yang ingin di-scan.

Opsi Tambahan

XSStrike memiliki banyak opsi tambahan, seperti:

  • --crawl: Melakukan crawling pada website.
  • --data "param1=value1¶m2=value2": Mengirimkan data POST.
  • --headers "Header1: Value1\nHeader2: Value2": Menambahkan custom headers.
  • --fuzzer: Menjalankan fuzzer untuk menemukan input yang vulnerable.

Untuk melihat daftar lengkap opsi, jalankan python3 xsstrike.py -h.

⚠️ Peringatan Penting

Perhatian! XSStrike adalah alat yang powerful. Gunakan dengan bijak dan bertanggung jawab.

  • Pastikan Anda memiliki izin sebelum melakukan scan terhadap website.
  • Jangan gunakan XSStrike untuk aktivitas ilegal.
  • Gunakan XSStrike hanya untuk tujuan pengujian keamanan dan pendidikan.

Penyalahgunaan XSStrike dapat melanggar hukum dan menyebabkan kerugian bagi orang lain!
