Sherlock adalah alat OSINT (Open Source Intelligence) yang digunakan untuk mencari profil media sosial berdasarkan username. Dengan Sherlock, kamu dapat melacak keberadaan username pada lebih dari 300 situs, membantu proses investigasi atau penelitian OSINT.

cd sherlock
python3 -m pip install -r requirements.txt

ğŸ” Penggunaan Dasar

Untuk menjalankan Sherlock, gunakan perintah berikut:

  • python3 sherlock.py - Melacak keberadaan username di berbagai platform.
  • python3 sherlock.py --csv - Menyimpan hasil dalam format CSV.
  • python3 sherlock.py --timeout 30 - Menentukan batas waktu setiap situs (dalam detik).
