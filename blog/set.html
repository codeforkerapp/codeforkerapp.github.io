Social Engineer Toolkit

📘 Apa itu Social Engineer Toolkit (SET)?

Social Engineer Toolkit (SET) adalah framework open-source yang dirancang untuk social engineering. SET menyediakan berbagai modul untuk melakukan berbagai serangan social engineering, seperti phishing, spear-phishing, dan serangan berbasis web. SET memungkinkan Anda untuk membuat email phishing yang meyakinkan, halaman web palsu, dan payload berbahaya untuk mencuri informasi sensitif.

🛠️ Instalasi

Termux

Informasi: SET tidak tersedia di repositori paket Termux. Kamu bisa mencoba menginstall SET secara manual dengan mengikuti panduan di GitHub SET. Namun, karena SET memiliki banyak dependensi, instalasi di Termux mungkin akan rumit dan membutuhkan konfigurasi tambahan.

Linux (Ubuntu/Debian)

sudo apt update
sudo apt install set -y

🔍 Penggunaan Dasar (Linux)

Memulai SET

setoolkit

Membuat Serangan Phishing

  1. Pilih opsi 1 (Social-Engineering Attacks).
  2. Pilih opsi 2 (Website Attack Vectors).
  3. Pilih opsi 3 (Credential Harvester Attack Method).
  4. Pilih opsi 2 (Site Cloner).
  5. Masukkan URL website yang ingin di-clone.
  6. SET akan membuat salinan website tersebut dan menyimpannya di direktori Anda.
  7. Kirimkan link website palsu tersebut ke target Anda.

⚠️ Peringatan Penting

Perhatian! SET adalah alat yang powerful. Gunakan dengan bijak dan bertanggung jawab.

  • Pastikan Anda memiliki izin sebelum melakukan serangan social engineering.
  • Jangan gunakan SET untuk aktivitas ilegal.
  • Gunakan SET hanya untuk tujuan pengujian keamanan dan pendidikan.

Penyalahgunaan SET dapat melanggar hukum dan menyebabkan kerugian bagi orang lain!
