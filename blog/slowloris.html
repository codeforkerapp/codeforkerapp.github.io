Slowloris

üìò Apa itu Slowloris?

Slowloris adalah tool untuk melakukan serangan Denial-of-Service (DoS) terhadap web server. Slowloris bekerja dengan mengirimkan HTTP request secara perlahan dan terus-menerus ke web server target, sehingga menghabiskan resource server dan membuatnya tidak dapat melayani request dari pengguna lain. Slowloris efektif karena hanya membutuhkan sedikit bandwidth dan dapat diluncurkan dari satu komputer.

üîç Penggunaan Dasar

Menjalankan Slowloris

python3 slowloris.py -v -s jumlah_socket -p 80 -u http://contoh.com

Keterangan:

  • -v: Mode verbose (menampilkan informasi detail).
  • -s: Jumlah socket yang digunakan (semakin banyak, semakin kuat serangan).
  • -p: Port target (default: 80).
  • -u: URL target.
