Nikto

📘 Apa itu Nikto?

Nikto adalah pemindai kerentanan web server open-source yang melakukan pengujian terhadap web server untuk mengidentifikasi berbagai masalah, termasuk versi server yang ketinggalan zaman, plugin yang tidak aman, dan konfigurasi yang berbahaya. Nikto juga dapat mendeteksi file dan direktori default yang tidak aman.

🛠️ Instalasi

Termux

Informasi: Saat ini, belum ada paket Nikto resmi yang tersedia di repositori Termux. Anda mungkin perlu mencoba metode instalasi alternatif, seperti kompilasi dari source code.

Linux (Ubuntu/Debian)

Update & Upgrade Sistem

sudo apt update
sudo apt upgrade -y

Install Nikto

sudo apt install nikto -y

🔍 Penggunaan Dasar

Cek Versi

nikto -v

Pindai Web Server

nikto -h http://contoh.com

Pindai dengan SSL

nikto -h https://contoh.com

Simpan Hasil Scan

nikto -h http://contoh.com -o hasil_scan.txt

⚠️ Peringatan Penting

Perhatian! Nikto adalah alat yang powerful. Gunakan dengan bijak dan bertanggung jawab.

  • Pastikan Anda memiliki izin sebelum memindai web server yang bukan milik Anda.
  • Jangan pernah menggunakan Nikto untuk aktivitas ilegal.
  • Gunakan Nikto hanya untuk tujuan pengujian keamanan dan pendidikan.

Penyalahgunaan Nikto dapat melanggar hukum dan menyebabkan kerugian bagi orang lain!
