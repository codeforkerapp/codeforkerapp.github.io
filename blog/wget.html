Wget

📘 Apa itu Wget?

Wget adalah command-line utility yang digunakan untuk mengunduh file dari internet. Wget mendukung berbagai protokol, termasuk HTTP, HTTPS, dan FTP. Wget sangat berguna untuk mengunduh file secara non-interaktif, sehingga Anda dapat mengotomatiskan proses pengunduhan.

🛠️ Instalasi

Termux

pkg update && pkg upgrade -y
pkg install wget -y

Linux (Ubuntu/Debian)

sudo apt update
sudo apt install wget -y

🔍 Penggunaan Dasar

Cek Versi

wget --version

Unduh File

wget https://contoh.com/file.zip

Unduh File dengan Nama Berbeda

wget -O nama_baru.zip https://contoh.com/file.zip

Lanjutkan Unduhan yang Terhenti

wget -c https://contoh.com/file.zip

Unduh di Background

wget -b https://contoh.com/file.zip

⚠️ Peringatan Penting

Perhatian! Meskipun Wget adalah alat yang sah, penting untuk menggunakannya dengan bijak dan bertanggung jawab.

  • Pastikan Anda mengunduh file hanya dari sumber yang tepercaya.
  • Hindari mengunduh file yang mencurigakan atau tidak dikenal.
  • Jangan gunakan Wget untuk melakukan aktivitas yang melanggar hukum atau merugikan orang lain.
