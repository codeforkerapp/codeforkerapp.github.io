Cupp

📘 Apa itu Cupp?

Cupp (Common User Passwords Profiler) adalah tool untuk membuat custom wordlist berdasarkan informasi tentang target. Cupp akan menanyakan beberapa pertanyaan tentang target, seperti nama, tanggal lahir, nama keluarga, hewan peliharaan, dan lain-lain. Kemudian, Cupp akan mengkombinasikan informasi tersebut untuk membuat wordlist yang berisi kemungkinan password yang digunakan oleh target.

🛠️ Instalasi

Termux

pkg update && pkg upgrade -y
pkg install python3 git -y
git clone https://github.com/Mebus/cupp.git
cd cupp

Linux (Ubuntu/Debian)

sudo apt update
sudo apt install python3 git -y
git clone https://github.com/Mebus/cupp.git
cd cupp

🔍 Penggunaan Dasar

Menjalankan Cupp

python3 cupp.py -i

Keterangan:

  • -i: Menjalankan Cupp dalam mode interaktif.

Cupp akan menanyakan beberapa pertanyaan tentang target. Jawablah pertanyaan tersebut untuk membuat wordlist yang spesifik untuk target.

⚠️ Peringatan Penting

Perhatian! Cupp adalah alat yang powerful. Gunakan dengan bijak dan bertanggung jawab.

  • Pastikan Anda mematuhi hukum dan etika saat menggunakan Cupp.
  • Jangan gunakan Cupp untuk membuat wordlist yang digunakan untuk aktivitas ilegal.
  • Gunakan Cupp hanya untuk tujuan pengujian keamanan dan pendidikan.

Penyalahgunaan Cupp dapat melanggar hukum dan menyebabkan kerugian bagi orang lain!
