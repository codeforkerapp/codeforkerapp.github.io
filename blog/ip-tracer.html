IP-Tracer

üìò Apa itu IP-Tracer?

IP-Tracer adalah alat yang digunakan untuk melacak alamat IP. Dikembangkan untuk Termux dan sistem berbasis Linux, IP-Tracer memanfaatkan API dari ip-api untuk mendapatkan informasi tentang alamat IP.

sudo apt update
sudo apt install git -y
git clone https://github.com/Th3-C0der/IP-Tracer.git
cd IP-Tracer
chmod +x install
sudo ./install

üîç Penggunaan Dasar

Setelah menginstal IP-Tracer, gunakan perintah berikut untuk menjalankan alat ini:

  • trace -s - Memulai menu utama IP-Tracer.
  • trace -m - Melacak alamat IP Anda sendiri.
  • trace -t [alamat-ip] - Melacak alamat IP target.
  • trace - Menampilkan informasi tambahan tentang alat ini.
