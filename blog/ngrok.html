Ngrok

üìò Apa itu Ngrok?

Ngrok adalah tool yang memungkinkan kamu membuat secure tunnels ke localhost. Dengan Ngrok, kamu bisa mengekspos server lokal yang berjalan di komputermu (misalnya, web server, SSH server) ke internet publik, sehingga bisa diakses dari mana saja. Ngrok sangat berguna untuk development, testing, dan demo aplikasi web tanpa perlu deployment ke server publik.

üîç Penggunaan Dasar

Membuat Tunnel ke HTTP Server (port 80)

./ngrok http 80

Membuat Tunnel ke HTTPS Server (port 443)

./ngrok http 443

Membuat Tunnel ke TCP Server (port custom)

./ngrok tcp 22  # Contoh untuk SSH server

‚ö†Ô∏è Peringatan Penting

Perhatian! Ngrok mengekspos server lokalmu ke internet publik. Gunakan dengan bijak dan bertanggung jawab.

  • Pastikan server lokalmu aman dan terlindungi sebelum diekspos ke internet.
  • Jangan mengekspos informasi sensitif atau layanan yang tidak seharusnya diakses publik.
  • Gunakan Ngrok hanya untuk keperluan development, testing, dan demo.
