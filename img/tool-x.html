Tool-X

📘 Apa itu Tool-X?

Tool-X adalah installer tool hacking untuk Termux. Tool-X memungkinkan Anda untuk menginstal berbagai tool hacking populer dengan mudah, seperti Nmap, Metasploit, Hydra, dan banyak lagi. Tool-X mengotomatiskan proses instalasi dan konfigurasi, sehingga Anda dapat dengan cepat memulai penetration testing di Termux.

🛠️ Instalasi

Termux

pkg update && pkg upgrade -y
pkg install git -y
git clone https://github.com/rajkumardusad/Tool-X.git
cd Tool-X
chmod +x install.aex
sh install.aex

Linux (Ubuntu/Debian)

Informasi: Tool-X dirancang khusus untuk Termux dan tidak tersedia untuk Linux. Kamu bisa menginstal tool-tool hacking secara manual di Linux menggunakan `apt`.

🔍 Penggunaan Dasar (Termux)

Memulai Tool-X

toolx

Setelah menjalankan perintah di atas, Anda akan melihat menu Tool-X. Anda dapat memilih tool yang ingin diinstal dengan memasukkan nomornya.

Menginstal Tool

  1. Masukkan nomor tool yang ingin diinstal.
  2. Tool-X akan mengunduh dan menginstal tool tersebut secara otomatis.
  3. Setelah selesai, Anda dapat menjalankan tool tersebut dengan mengetikkan namanya di Termux.
