Knockpy adalah tool untuk melakukan enumerasi subdomain dengan cepat pada target domain melalui passive reconnaissance dan dictionary scan. Tool ini berguna untuk menemukan subdomain tersembunyi yang mungkin tidak terdeteksi dengan metode lain. Informasi ini dapat digunakan untuk pemetaan jaringan, penetration testing, dan analisis keamanan.

ğŸ” Penggunaan Dasar

Menjalankan Knockpy

python3 knockpy.py domain.com

Keterangan:

  • Ganti domain.com dengan nama domain target.

Opsi Tambahan

Knockpy memiliki beberapa opsi tambahan, seperti:

  • --wordlist wordlist.txt: Menggunakan wordlist kustom.
  • --silent: Menjalankan scan dalam mode silent.
  • --output output.txt: Menyimpan hasil scan ke file.

Untuk melihat daftar lengkap opsi, jalankan python3 knockpy.py -h.
