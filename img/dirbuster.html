Dirbuster

📘 Apa itu Dirbuster?

Dirbuster adalah tool web content scanner berbasis Java yang digunakan untuk menemukan direktori dan file tersembunyi di website. Dirbuster melakukan brute-force attack dengan mengirimkan request ke web server dan menganalisis respon yang diterima. Dirbuster sangat berguna untuk penetration testing, karena dapat membantu menemukan direktori admin, file backup, dan informasi sensitif lainnya yang mungkin tidak terlihat secara langsung.

🛠️ Instalasi

Termux

Informasi: Dirbuster tidak tersedia di repositori paket Termux. Dirbuster membutuhkan Java Runtime Environment (JRE) untuk dijalankan. Sebagai alternatif, Anda dapat menggunakan tool lain seperti Gobuster atau Feroxbuster.

Gobuster

pkg update && pkg upgrade -y
pkg install gobuster -y

Feroxbuster

pkg update && pkg upgrade -y
pkg install feroxbuster -y

Linux (Ubuntu/Debian)

sudo apt update
sudo apt install dirbuster -y

🔍 Penggunaan Dasar (Linux)

Menjalankan Dirbuster

dirbuster

Setelah menjalankan perintah di atas, GUI Dirbuster akan muncul. Kamu perlu mengkonfigurasi beberapa opsi, seperti:

  • Target URL: Masukkan URL website yang ingin di-scan.
  • Wordlist: Pilih file wordlist yang berisi daftar direktori dan file untuk dicoba.
  • Threads: Jumlah thread yang digunakan untuk melakukan scan (semakin banyak thread, semakin cepat proses scan, tetapi juga semakin membebani web server).

⚠️ Peringatan Penting

Perhatian! Dirbuster adalah alat yang powerful. Gunakan dengan bijak dan bertanggung jawab.

  • Pastikan Anda memiliki izin sebelum melakukan scan terhadap website.
  • Jangan gunakan Dirbuster untuk aktivitas ilegal.
  • Gunakan Dirbuster hanya untuk tujuan pengujian keamanan dan pendidikan.

Penyalahgunaan Dirbuster dapat melanggar hukum dan menyebabkan kerugian bagi orang lain!
