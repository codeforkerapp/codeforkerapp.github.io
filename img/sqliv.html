SQLiv

📘 Apa itu SQLiv?

SQLiv adalah mass SQL injection vulnerability scanner yang dapat digunakan untuk menemukan website yang rentan terhadap serangan SQL injection. SQLiv dapat melakukan scan terhadap banyak website sekaligus, baik dengan menggunakan dork SQL injection maupun dengan melakukan crawling pada website target.

🛠️ Instalasi

Termux

pkg update && pkg upgrade -y
pkg install python3 git -y
git clone https://github.com/Code-L0V3R/sqliv.git
cd sqliv
pip3 install -r requirements.txt

Linux (Ubuntu/Debian)

sudo apt update
sudo apt install python3 python3-pip git -y
git clone https://github.com/Code-L0V3R/sqliv.git
cd sqliv
pip3 install -r requirements.txt

🔍 Penggunaan Dasar

Single Target Scan

python3 sqliv.py -t http://www.example.com

Multiple Target Scan (dengan Dork)

python3 sqliv.py -d "inurl:index.php?id="

Opsi Tambahan

SQLiv memiliki banyak opsi tambahan, seperti:

  • -c: Jumlah thread yang digunakan untuk crawling.
  • -v: Menampilkan informasi verbose.
  • -o: Menyimpan hasil scan ke file.

Untuk melihat daftar lengkap opsi, jalankan python3 sqliv.py -h.

⚠️ Peringatan Penting

Perhatian! SQLiv adalah alat yang powerful. Gunakan dengan bijak dan bertanggung jawab.

  • Pastikan Anda memiliki izin sebelum melakukan scan terhadap website.
  • Jangan gunakan SQLiv untuk aktivitas ilegal.
  • Gunakan SQLiv hanya untuk tujuan pengujian keamanan dan pendidikan.

Penyalahgunaan SQLiv dapat melanggar hukum dan menyebabkan kerugian bagi orang lain!
