Tsunami

📘 Apa itu Tsunami?

Tsunami adalah network security scanner yang dikembangkan oleh Google. Tool ini dapat mendeteksi berbagai kerentanan keamanan pada jaringan, seperti open port yang tidak aman, service yang rentan, dan konfigurasi yang salah. Tsunami menggunakan plugin system yang extensible, sehingga mudah dikonfigurasi dan disesuaikan dengan kebutuhan.

🛠️ Instalasi

Termux

Informasi: Tsunami tidak tersedia di repositori paket Termux. Kamu mungkin perlu mencoba metode instalasi alternatif, seperti mengunduh source code dari GitHub dan melakukan kompilasi manual.

Linux (Ubuntu/Debian)

Tsunami membutuhkan nmap dan ncrack. Pastikan kamu sudah menginstalnya:

sudo apt update
sudo apt install nmap ncrack -y

Kemudian, jalankan script instalasi Tsunami:

bash -c "$(curl -sfL https://raw.githubusercontent.com/google/tsunami-security-scanner/master/install.sh)"

🔍 Penggunaan Dasar

Melakukan Scan

tsunami scan -n 192.168.1.0/24 -p 80,443

Keterangan:

  • -n: Network atau IP address yang ingin di-scan.
  • -p: Port yang ingin di-scan.

Opsi Tambahan

Tsunami memiliki banyak opsi tambahan, seperti:

  • -r: Path ke file hasil scan.
  • -v: Menampilkan informasi verbose.

Untuk melihat daftar lengkap opsi, jalankan tsunami -h.

⚠️ Peringatan Penting

Perhatian! Tsunami adalah alat yang powerful. Gunakan dengan bijak dan bertanggung jawab.

  • Pastikan Anda memiliki izin sebelum melakukan scan terhadap jaringan.
  • Jangan gunakan Tsunami untuk aktivitas ilegal.
  • Gunakan Tsunami hanya untuk tujuan pengujian keamanan dan pendidikan.

Penyalahgunaan Tsunami dapat melanggar hukum dan menyebabkan kerugian bagi orang lain!
