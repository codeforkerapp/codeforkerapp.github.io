Yersinia

📘 Apa itu Yersinia?

Yersinia adalah framework untuk melakukan serangan layer 2 pada jaringan. Yersinia didesain untuk memanfaatkan kelemahan pada berbagai protokol jaringan, seperti Spanning Tree Protocol (STP), Cisco Discovery Protocol (CDP), dan Dynamic Trunking Protocol (DTP). Yersinia dapat digunakan untuk melakukan network reconnaissance, denial of service, dan man-in-the-middle attacks.

🛠️ Instalasi

Termux

Informasi: Yersinia tidak tersedia di repositori paket Termux.

Linux (Ubuntu/Debian)

sudo apt update
sudo apt install yersinia -y

🔍 Penggunaan Dasar (Linux)

Memulai Yersinia

yersinia -G

Keterangan:

  • -G: Menjalankan Yersinia dalam mode GUI (Graphical User Interface).

Setelah Yersinia berjalan, kamu bisa memilih protokol yang ingin diserang dan mengkonfigurasi opsi serangan sesuai kebutuhan.

⚠️ Peringatan Penting

Perhatian! Yersinia adalah alat yang powerful. Gunakan dengan bijak dan bertanggung jawab.

  • Pastikan Anda memiliki izin sebelum melakukan serangan terhadap jaringan.
  • Jangan gunakan Yersinia untuk aktivitas ilegal.
  • Gunakan Yersinia hanya untuk tujuan pengujian keamanan dan pendidikan.

Penyalahgunaan Yersinia dapat melanggar hukum dan menyebabkan kerugian bagi orang lain!
