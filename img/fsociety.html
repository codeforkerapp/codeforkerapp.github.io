Fsociety

📘 Apa itu Fsociety?

Fsociety adalah penetration testing framework yang terinspirasi dari serial TV Mr. Robot. Framework ini menyediakan berbagai tool untuk information gathering, password attack, wireless testing, exploitation, sniffing & spoofing, dan web hacking. Fsociety memudahkan pengguna untuk melakukan penetration testing dengan menyediakan antarmuka yang user-friendly.

🛠️ Instalasi

Termux

pkg update && pkg upgrade -y
pkg install git python2 -y
git clone https://github.com/Manisso/fsociety.git
cd fsociety
pip install -r requirements.txt

Linux (Ubuntu/Debian)

sudo apt update
sudo apt install git python3 python3-pip -y
git clone https://github.com/Manisso/fsociety.git
cd fsociety
pip3 install -r requirements.txt

🔍 Penggunaan Dasar

Menjalankan Fsociety

python3 fsociety.py

Setelah dijalankan, Fsociety akan menampilkan menu dengan berbagai kategori tool. Kamu bisa memilih kategori dan tool yang ingin digunakan.

⚠️ Peringatan Penting

Perhatian! Fsociety adalah alat yang powerful. Gunakan dengan bijak dan bertanggung jawab.

  • Pastikan Anda memiliki izin sebelum menggunakan tool hacking.
  • Jangan gunakan Fsociety untuk aktivitas ilegal.
  • Gunakan Fsociety hanya untuk tujuan pengujian keamanan dan pendidikan.

Penyalahgunaan Fsociety dapat melanggar hukum dan menyebabkan kerugian bagi orang lain!
