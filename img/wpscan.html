WPScan adalah black box WordPress vulnerability scanner yang digunakan untuk memindai situs web WordPress dan mengidentifikasi potensi celah keamanan. WPScan dapat mendeteksi versi WordPress yang rentan, plugin dan tema yang tidak aman, konfigurasi yang lemah, dan informasi sensitif lainnya. WPScan sangat berguna untuk penetration testing dan audit keamanan website WordPress.

Termux

Informasi: WPScan membutuhkan Ruby untuk dijalankan. Meskipun Ruby bisa diinstall di Termux, menginstall WPScan di Termux bisa jadi rumit karena dependensi dan konfigurasi yang diperlukan. Sebagai alternatif, Anda dapat menggunakan tool online seperti WPScan Online atau mencoba menginstalnya secara manual dengan mengikuti panduan di GitHub WPScan.

Linux (Ubuntu/Debian)

sudo apt update
sudo apt install ruby ruby-dev git -y
git clone https://github.com/wpscanteam/wpscan.git
cd wpscan
sudo gem install bundler && bundle install --without test
