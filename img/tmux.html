Tmux adalah terminal multiplexer. Tmux memungkinkan kamu untuk membuat, mengakses, dan mengontrol multiple terminal sessions dari satu layar. Dengan Tmux, kamu bisa menjalankan proses di background, berpindah antar sessions dengan mudah, dan membagi layar menjadi beberapa panes. Tmux sangat berguna untuk multitasking dan mengelola banyak aplikasi command-line secara efisien.

ğŸ” Penggunaan Dasar

Memulai Tmux Session

tmux

Membuat Window Baru

Ctrl+b c

Berpindah Antar Window

Ctrl+b n  (window selanjutnya)
Ctrl+b p  (window sebelumnya)

Membagi Layar (Pane)

Ctrl+b "  (membagi secara horizontal)
Ctrl+b %  (membagi secara vertikal)

Berpindah Antar Pane

Ctrl+b arah panah

Melepas Tmux Session (Detach)

Ctrl+b d

Menyambung Kembali ke Tmux Session (Attach)

tmux attach
