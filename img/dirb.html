Dirb adalah tool web content scanner yang digunakan untuk menemukan direktori dan file tersembunyi di website. Dirb melakukan ini dengan mengirimkan request ke web server dan menganalisis respon yang diterima. Dirb sangat berguna untuk penetration testing, karena dapat membantu menemukan direktori admin, file backup, dan informasi sensitif lainnya yang mungkin tidak terlihat secara langsung.
