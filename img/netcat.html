Netcat

📘 Apa itu Netcat?

Netcat, atau sering disingkat `nc`, adalah command-line utility yang serbaguna untuk membaca dan menulis data melalui koneksi jaringan TCP atau UDP. Netcat sering disebut sebagai "Swiss Army knife" untuk jaringan karena kemampuannya yang luas, termasuk port scanning, transfer file, dan pembuatan backdoor.

🛠️ Instalasi

Termux

pkg update && pkg upgrade -y
pkg install netcat -y

Linux (Ubuntu/Debian)

sudo apt update
sudo apt install netcat -y

🔍 Penggunaan Dasar

Cek Versi

nc -h

Membuat Koneksi TCP

nc -v ip_address port

Mendengarkan Koneksi TCP

nc -l -p port

Transfer File

# Pada pengirim:
nc -v ip_penerima port < file.txt

# Pada penerima:
nc -l -p port > file.txt

Port Scanning

nc -v -z -w 1 ip_address 1-1000

⚠️ Peringatan Penting

Perhatian! Netcat adalah alat yang powerful. Gunakan dengan bijak dan bertanggung jawab.

  • Pastikan Anda memiliki izin sebelum terhubung ke sistem lain.
  • Jangan gunakan Netcat untuk aktivitas ilegal.
  • Pahami risiko keamanan yang terkait dengan penggunaan Netcat, terutama saat membuat backdoor.

Penyalahgunaan Netcat dapat melanggar hukum dan membahayakan sistem Anda!
